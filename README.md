# Real-time Collaborative Code Editor

- Real-time multi-user editing
- Live cursor sync
- WebSocket-based communication
- Code execution sandbox (Docker)

## Tech Stack
- React + Monaco Editor
- Node.js + Express
- WebSockets
- Docker
